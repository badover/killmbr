# killmbr
Simple way to overwrite mbr with python

## Usage
### Preparation and compiling
#### Install pywin32 library
```bash
pip install pywin32 
```
#### Intsall auto-py-to-exe
```bash
pip install auto-py-to-exe
```
### Convert to exe
```bash
auto-py-to-exe
```
##### Might not work in some VMs!
